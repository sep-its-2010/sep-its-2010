\subsection{Detailed Description}
The \hyperlink{classsep_1_1conquest_1_1model_1_1_drive_request}{DriveRequest} class is a message object for \hyperlink{interfacesep_1_1conquest_1_1model_1_1_i_com_client}{IComClient} clients. It represents a drive command for a specific robot. Therefore the command will be returned by the getCommand method.
