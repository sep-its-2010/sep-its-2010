\subsection{Detailed Description}
The class \hyperlink{classsep_1_1conquest_1_1model_1_1_environment}{Environment} represents the model corresponding to the Model-\/View-\/ Controller-\/pattern. It is used as interface between robots and the graphical user interface. Every {\ttfamily Activity} has to register in order to be notified.
